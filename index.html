Get your code on the interwebs

1. Sign up for these things

  1. Github
  2. Codepen (you can sign up to this with your new Github account)

2. Create a repository in Github

Name it "[your github username].github.io"

Then, make a couple of files in your new repo and name them:

  • index.html
  • style.css

3. Enable Github pages

In Github, click the settings cog. Then click "pages". Then [todo]

This will put your new (empty) site on the internet

4. Add this code to index.html

  <!DOCTYPE html>
<html lang="en">
  
<head>
  <meta charset="UTF-8">
  <title>Change me</title>
  <link rel="stylesheet" href="style.css">
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>

<body>
  <!-- Your code goes here! --> 
</body>

</html>

5. We're ready to start coding!

Head over to Codepen and play with the html and css files

Paste what you wrote into your Github repo and it'll appear on the internet a few minutes later
